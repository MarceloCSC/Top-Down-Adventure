# Top-Down Adventure Prototype

Prototype of a 2D top-down adventure game developed in Unity, inspired by classic games like *The Legend of Zelda*.

## Overview

This project explores the development of core systems for a top-down adventure game, including player movement, interaction, and basic combat mechanics.

Although not a complete game, it serves as a foundation for experimenting with gameplay structure and system design in a 2D environment.

## Features

- Top-down player movement and directional animation  
- Basic combat / attack system  
- Interaction system for world elements  
- State-driven player behavior  
- Foundation for exploration-based gameplay  

## Tech Stack

- Unity (2D)
- C#

## Purpose

The project was created as an experimental prototype to explore how classic top-down adventure mechanics can be implemented in Unity, focusing on structure and gameplay systems rather than content completeness.
