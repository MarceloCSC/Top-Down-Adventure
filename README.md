# Top-Down Adventure Prototype

Prototype of a 2D top-down action-adventure game developed in Unity, inspired by classic games like *The Legend of Zelda*.

## Overview

This project explores the implementation of core systems for a top-down action-adventure game, including player movement, melee combat, enemy behaviors, world interaction, pickups, and scene progression.

Although it was never developed into a complete game, the project works as a solid gameplay prototype and a foundation for experimenting with interconnected 2D systems.

## Systems Implemented

- Top-down player movement with directional animation  
- Melee combo / dash attack system  
- Combat aiming with enemy target snapping  
- Player health, mana, stamina, and knockback systems  
- Enemy AI with patrol, chase, attack, stun, and death states  
- Projectile-based enemy behavior  
- Inventory system with items, keys, and coins  
- Interactable world objects such as doors, chests, readable objects, and breakables  
- Loot and stat pickups (health, mana, coins)  
- Room transfer and scene transition systems  
- Basic player data persistence  

## Tech Stack

- Unity (2D)  
- C#  

## Purpose

The project was created as a prototype to explore how the core systems of a top-down action-adventure game could be structured and implemented in Unity, with a focus on gameplay systems and extensibility rather than content completeness.
